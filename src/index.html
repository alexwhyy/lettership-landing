Power to the writers.

Lettership is an open-source blogging and newsletter platform for writers.

Simplicity as a core principle.

Powerful functionality without over-engineering features. No need to constantly update and install new plugins.

The best writing experience.

A clean interface for you to write with no distractions and a powerful editor with rich text support.

Make it your own with themes.

Lettership comes with multiple prebuilt themes. Developers can use Lettership's templating system to create new themes with just a bit of HTML and CSS.

Hardened and modern security.

Lettership is building the most secure and open publishing platform. We're paranoid with security and follow modern security practices.

Go open-source with zero lock-in.

All of your subscriber and post data is openly available to you. Your data is in your control.
